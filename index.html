About

Martlet Solutions LLC

Martlet Solutions LLC is an independent Texas-based software company building AI tools for programming education.

We ship two products — CodeTeach.ai and StudySite.ai — used by instructors teaching college-level programming courses. Both are built on the same stack: Next.js, sandboxed cloud code execution, and the OpenAI Agents SDK.

AI-Generated Assignments

CodeTeach.ai produces a complete GitHub Classroom assignment — starter code, solution, tests, README, and autograding workflow — deployed atomically as a template repo in your GitHub org.

Browser-Based Code Sandbox

StudySite.ai lets students write and run real code in 12+ languages — Python, Java, C++, Rust, Go, SQL, Jupyter and more — with no local setup.

GitHub App Integration

Both products plug into instructors' existing GitHub Classroom organizations. Student code lives in Git; assignments deploy as template repositories.

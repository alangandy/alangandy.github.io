CodeTeach.ai — AI-Generated Programming Assignments in Minutes

Instructors spend hours authoring GitHub Classroom assignments — writing starter code, solutions, tests, READMEs, and autograding workflows by hand. CodeTeach.ai compresses that into a 3-step wizard powered by your choice of AI model. Describe the assignment, review the generated code, deploy the template repo.

1. Seed 2. Review 3. Deploy
  • ✓ Generates starter code, solution, tests, README, and GitHub Actions autograding workflow — all consistent and validated
  • ✓ 12 languages: Python, Java, JavaScript, TypeScript, C++, C, C#, Go, Rust, Ruby, SQL, Jupyter notebooks
  • ✓ Sandboxed validation in E2B before deploy — broken assignments never reach students
  • ✓ Bring your own API key — OpenAI, Anthropic, Gemini, Mistral, Groq, or DeepSeek
  • ✓ Atomic deploy via GitHub App — single-commit template repo in your org, ready for Classroom

## Can I revisit a previous page or question?
Please do not use your browser's 'back button.' The underlying logic of the application may trigger data loss if a user navigates with browser buttons.

Users should follow the path presented on each screen, utilizing navigation buttons within the application.

If a user wishes to revisit a preious page, (while within an interview,) the user should use the *Review Answers* link in the upper right navigation menu. *This menu is available by clicking the user's email address in the upper right area of the application.*
